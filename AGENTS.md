# Цель проекта: создание удобного тг бота для уведомлений когда у кого-то в группе происходит др

организация:

1. Сбор данных.
   Вручную вводить фио и др всех одногруппников очень не удобно, поэтому важно упростить этот сценарий.
   Данные собираются путём перебора id в web лмс вышки (там находится ФИО), потом по каждому из ФИО запускается поиск
   через api мобильного приложения (по ФИО), берётся первый результат (там есть ФИО, дата рождения и группа + ещё часть
   данных). Все эти данные сохраняются в бд

2. Сам тг бот, функционал довольно базовый: добавление по группе (например БПМИ25 добавляет всех кто поступил на ПМИ в
   25 году, БПМИ251 добавляет только всю первую группу) можно добавлять несколько групп. Возможность добавления по фио (
   например академ рука или какого-то другого человека, который важен для группы) с хорошим поиском (например часто
   полное фио не известно, но известно Имя Фамилия и что это преподаватель), важно чтобы этот процесс добавления был
   удобным

# Желаемый вид проекта

Написан на современном kotlin, с использованием ktor, kotlinx serialization, exposed, koin, postgresql, jsoup, eu.vendeli:telegram-bot (github).
Качество кода должно на хорошем уровне. Читаемость кода очень важна, на одной строчке не должно быть слишком много сложного кода. Например в одном файле не должно быть больше 300 строк, а функции не должны быть слишком длинными (100 и более строк), можно ориентироваться на правила detekt, хоть и не требуется их полностью соблюдать.
Полные импорты должны использовать тогда когда это реально нужно. Используй разные logger для разных модулей. Переиспользуй код, вынося его в функции и extensions.
